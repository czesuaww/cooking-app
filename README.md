Cooking App is a web application for browsing and managing cooking recipes.
The project is currently work in progress and focuses on the frontend layer.

The goal of the application is to provide a clean and user-friendly interface that allows users to:
browse cooking recipes, organize them in a clear and structured way, easily extend the project with backend functionality in the future.

Tech Stack:
Frontend:
-	React
-	JavaScript
-	HTML
-	CSS

Planned Backend (future development):
-	PHP
-	Database integration (e.g. MySQL)
  
Getting Started:
To run the project locally:
- npm install
- npm run dev

Project Status:
This project is under active development.
New features such as database connection, backend API, and user interaction will be added in future updates.
